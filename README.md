# Tic Tac Toe in Python

A simple command-line Tic Tac Toe game developed in Python.

## Features

* Play against a computer opponent
* Choose your marker (X or O)
* Randomized first turn selection
* Win, lose, and draw detection
* Input validation for player moves
* Replay option after each game

## Technologies Used

* Python 3
* Random module

## How to Run

1. Clone the repository:
   git clone https://github.com/NiravBaghel/Tictactoe.py/blob/main/TTT.py

2. Navigate to the project directory:
   cd TTT-python

3. Run the game:
   python TTT.py

Enjoy playing Tic Tac Toe against the Master!
